# NCMDforPwsh
网易云音乐歌曲下载器PowerShell实现，通过[Wget for Windows](https://eternallybored.org/misc/wget/)进行下载

# 使用方法:

方法1：右键选择`使用PowerShell运行`，输入歌曲ID

方法2：打开PowerShell，输入
```
iwr -uri "https://raw.githubusercontent.com/LGY07/NCMDforPwsh/main/ncmd.ps1" -out ncmd.ps1
powershell .\ncmd.ps1
```
然后输入歌曲ID
